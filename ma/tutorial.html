MA Install & Usage Tutorial

How to Use MA

After installing MA to /usr/local/bin/ma, you can run it directly from your terminal:

[toby@archlinux .bin]$ ma
Usage: ma <command> [args]

Available commands:
  (none defined yet â€” use 'ma set')

Special commands:
  ma set <name> <command>    # add or overwrite a shortcut
  ma del <name>                  # delete a shortcut
  ma list                              # show all commands
  ma help                              # show this menu
[toby@archlinux .bin]$

You can define your own shortcuts with ma set <name> <command>, remove them with ma del <name>, and list all shortcuts with ma list.
